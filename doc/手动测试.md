# 启动服务器

```bash
cd objstore && go build main.go && cd ../
# 启动数据节点
LISTEN_ADDRESS="data1:8080" STORAGE_ROOT=storage_root/ RABBITMQ_SERVER="amqp://objs:objs@192.168.0.108:5672/objs" IS_DATA_NODE=true objstore/main
# 启动 API 节点
LISTEN_ADDRESS="data1:8080" STORAGE_ROOT=storage_root/ RABBITMQ_SERVER="amqp://objs:objs@192.168.0.108:5672/objs" IS_DATA_NODE=false objstore/main
```
